# Cawfee House

Cawfee House is a place for people to track grind sizes, brew times, temperatures, beans, and more for their favorite coffees. Store your favorite recipes!

## Technologies

### Front End

<ul>
<li>React</li>
<li>Material UI</li>
</ul>

### Back End

<ul>
<li>Node</li>
<li>Express</li>
<li>PostgreSQL</li>
</ul>

### Hosting

<ul>
<li>Firebase / Heroku</li>
<li>Netlify</li>
</ul>
